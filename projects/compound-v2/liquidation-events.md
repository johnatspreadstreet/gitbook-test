# Liquidation Events

### SAMPLE USAGE

`=COMPOUND_V2_LIQUIDATION_EVENTS(<field>,<name>)`

### SYNTAX

`=COMPOUND_V2_LIQUIDATION_EVENTS(input_range,num_rows)`

* input\_range - The range to constrain
* num\_rows - The number of rows the result should contain
